# Networking Fundamentals
**Platform:** TryHackMe  
**Path:** SOC Level 1 / Pre-Security  
**Difficulty:** Easy  

---

## Overview

This room covers the foundational networking concepts that every SOC analyst needs to understand -- how data travels across networks, key protocols, and how devices communicate. No hands-on lab for this room; content is theoretical.

---

## Key Concepts Covered

- OSI Model and its 7 layers
- TCP/IP Model
- IP addressing (IPv4/IPv6)
- Ports and protocols (HTTP, HTTPS, FTP, SSH, DNS)
- How packets travel across networks
- Subnetting basics

---

## Key Takeaways

- Understanding networking fundamentals is essential for analyzing traffic and identifying anomalies in a SOC environment.
- Knowing which ports and protocols are standard helps analysts spot unusual or suspicious connections quickly.
